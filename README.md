# Instalation:
Create virtual environment:

`python3 -m venv venv`

Activate venv:

`source venv/bin/activate`

Install dependencies:

`pip install -r requirements.txt`

Run migration:

`./manage.py migrate`

Run server:

`./manage.py runserver {HOST}:{PORT}`



# Documentation:

Documentation can be found here:

http://{HOST}/swagger/
